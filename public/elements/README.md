# Element Images

Place your element symbol images in this folder with the following naming convention:

- `Cyro.png` - Cyro element (cyan snowflake/star)
- `Electro.png` - Electro element (purple triskelion)
- `Pyro.png` - Pyro element (red flame)
- `Hydro.png` - Hydro element (cyan water/waves)
- `Dendro.png` - Dendro element (teal plant/floral)
- `Geo.png` - Geo element (golden yellow rhombus)
- `Anemo.png` - Anemo element (optional)

The images will automatically display on the GenshinFeature screen for element cards.

Supported formats: PNG, JPG, SVG, WEBP
